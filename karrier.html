Tanulási út részletesen

A Scrimba Front-End Developer karrierút egy 70+ órányi interaktív, projektalapú tanulási program, ahol minden készséget valós feladatokon keresztül sajátítottam el.

1. Alapszint – Web Dev Basics & CSS

HTML5 szemantikus elemek, CSS3 alapok és box model, responsive mobile-first gondolkodás, Flexbox és CSS Grid bevezető.
Projektek: Living the Simple Life blog, BBQ Splash Page, reszponzív portfolió oldalak.

2. Középhaladó – Interactive JavaScript

Modern ES6+ szintaxis, DOM-manipuláció, aszinkron művezetek (fetch, async/await), Promise-kezelés.
Projektek: Blackjack játék, Chrome Dashboard bővítmény, Pac-Man prototípus, modal komponensek.

3. Haladó – UI Design & Advanced Layouts

UI Design alapelvek (white space, kontraszt, tipográfia), komplex Grid és Flexbox layout-ok, Tailwind CSS.
Projektek: Photo Gallery, Pokedex, Jeopardy játék, responsive könyvtár alkalmazás.

4. React & Modern Development

Funkcionális komponensek, hooks (useState, useEffect), React Router 6, állapotkezelés, komponens kompozíció.
Projektek: Todo App többszakaszos fejlesztés, blog alkalmazás bejelentkező rendszerrel, Pokedex React integráció.
